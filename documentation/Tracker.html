Class: Tracker

Tracker(optionsopt)

Tracker class provides the basic logic to extend Newrelic's Browser Agent capabilities. Trackers are designed to listen third party elements (like video tags, banners, etc.) and send information over to Browser Agent. Extend this class to create your own tracker, override registerListeners and unregisterListeners for full coverage!

sendHeartbeat(attopt)

Heartbeating allows you to call this function each X milliseconds, defined by getHeartbeat. This is useful to send regular events to track changes. By default it will send Tracker.Events.HEARTBEAT. To start heartbeating use startHeartbeat and to stop them use stopHeartbeat.
Parameters:
Name Type Attributes Description
att Object <optional>
 Collection of key:value attributes to send with the request.
